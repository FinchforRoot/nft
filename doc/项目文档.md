功能说明：
这是一个NFT拍卖合约项目，核心功能实现了：
1.创建拍卖：允许用户将 NFT 上架拍卖
2.出价：允许用户以 ERC20 或以太坊出价。
3.结束拍卖：拍卖结束后，NFT 转移给出价最高者，资金转移给卖家。
4.撤销拍卖：允许用户在拍卖未开始前撤销自己创建的拍卖。


部署步骤：
1: 部署 TestAggregator (ETH/USD)
2: 部署 TestAggregator (USDT/USD)
3: 部署 TestERC20
4: 部署 TestMyNft
5: 部署 NftAuction (Implementation)
6: 部署 ERC1967Proxy
7: 通过 Proxy 调用 initialize()
8: 通过 Proxy 调用 setPriceFeed(address(0), ethAggregator)
9: 通过 Proxy 调用 setPriceFeed(usdtAddress, usdtAggregator)
10: 调用 usdt.mint(deployer, 1000 USDT)
11: 调用 myNft.mint(deployer)
12: 结束
